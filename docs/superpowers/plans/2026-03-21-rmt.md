## Phase 2: Core Modules

### src/data.py
- `generate_modular_addition(p=97, seed=42)` -> (X_train, y_train, X_test, y_test)
  - Input: pairs (a, b) one-hot encoded to 2*p dims
  - Output: (a+b) mod p as class label
  - 70/30 train/test split
- `generate_regression(n_samples=1000, seed=42)` -> (X_train, y_train, X_test, y_test)
  - Input: x values uniformly in [-1, 1], expanded to polynomial features [x, x^2, x^3]
  - Output: 0.5*x^3 - 0.3*x^2 + 0.7*x - 0.1

### src/model.py
- `class TinyMLP(nn.Module)`: 3 linear layers with ReLU
  - `__init__(input_dim, hidden_dim, output_dim)`
  - `forward(x)` -> logits
  - `get_weight_matrices()` -> list of (name, weight_tensor) tuples

### src/train.py
- `train_model(model, X, y, task, epochs, lr, seed)` -> training history
  - task = "classification" or "regression"
  - Returns: dict with loss_history, final_accuracy/final_mse
  - Sets torch manual_seed, deterministic mode

### src/rmt_analysis.py
- `marchenko_pastur_pdf(x, gamma, sigma_sq)` -> density values
- `marchenko_pastur_cdf(x, gamma, sigma_sq)` -> CDF values
- `compute_mp_bounds(gamma, sigma_sq)` -> (lambda_minus, lambda_plus)
- `analyze_weight_matrix(W)` -> dict with:
  - eigenvalues, gamma, sigma_sq, lambda_bounds
  - ks_statistic, ks_pvalue
  - outlier_fraction
  - spectral_norm_ratio
  - kl_divergence (binned)

### src/plots.py
- `plot_eigenvalue_spectra(results, save_path)` -> saves PNG
- `plot_ks_summary(results, save_path)` -> saves PNG

### src/report.py
- `generate_report(results)` -> markdown string
  - Summary table of KS stats per layer/width/task
  - Key findings

## Phase 3: Integration

### run.py
- Working-directory guard (check for SKILL.md)
- Step 1: Generate datasets
- Step 2: Train models (4 widths x 2 tasks = 8 models)
- Step 3: Analyze weight matrices (trained + untrained baselines)
- Step 4: Generate plots
- Step 5: Save results and report

### validate.py
- Load results.json
- Check all expected models present
- Check metric ranges (KS in [0,1], outlier_fraction in [0,1])
- Verify trained KS > untrained KS (core hypothesis)
- Error accumulation pattern

## Phase 4: Tests

### tests/test_data.py
- Test modular addition output shapes and ranges
- Test regression output shapes

### tests/test_model.py
- Test MLP forward pass shapes
- Test get_weight_matrices returns correct count

### tests/test_rmt_analysis.py
- Test MP bounds formula
- Test that random matrix has low KS statistic
- Test that structured matrix has high KS statistic
