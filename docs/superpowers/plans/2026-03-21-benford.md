## Implementation Steps

### Step 1: src/data.py
- `generate_modular_data(p=97, seed=42)` -> (X_train, y_train, X_test, y_test)
  - Input: pairs (a, b) as float tensors scaled to [0, 1]
  - Output: (a + b) mod p as class labels
  - Train/test split: 70/30
- `generate_sine_data(n=1000, seed=42)` -> (X_train, y_train, X_test, y_test)
  - Input: x ~ U(0, 2*pi)
  - Output: sin(x)
  - Train/test split: 70/30

### Step 2: src/model.py
- `class TinyMLP(nn.Module)`:
  - `__init__(self, d_in, d_hidden, d_out, n_layers=2)`
  - Uses ReLU activation
  - Returns named parameter dict for per-layer analysis

### Step 3: src/train.py
- `train_model(model, X_train, y_train, task_type, epochs=5000, lr=1e-3, snapshot_epochs=[0, 100, 500, 1000, 2000, 5000], seed=42)`
  - Returns: dict of {epoch: state_dict_copy} + loss history
  - Uses CrossEntropyLoss for classification, MSELoss for regression
  - Sets all seeds (torch, numpy, random)

### Step 4: src/benford_analysis.py
- `benford_expected()` -> dict {1: 0.301, 2: 0.176, ...}
- `extract_leading_digits(weights: np.ndarray)` -> np.ndarray of digits 1-9
  - Takes absolute value, excludes zeros and tiny values (<1e-10)
  - Extracts leading significant digit
- `compute_digit_distribution(digits: np.ndarray)` -> dict {1: freq, ...}
- `chi_squared_test(observed: dict, n: int)` -> (chi2_stat, p_value, df=8)
- `mad_from_benford(observed: dict)` -> float
  - MAD = (1/9) * sum |obs_d - exp_d|
- `kl_divergence(observed: dict)` -> float
- `analyze_snapshot(state_dict, layer_names=None)` -> per-layer and aggregate results
- `classify_mad(mad_value)` -> "close" | "acceptable" | "marginal" | "nonconformity"

### Step 5: src/plots.py
- `plot_digit_distribution(observed, title, save_path)` -- bar chart with Benford overlay
- `plot_mad_over_training(results_by_epoch, save_path)` -- line plot
- `plot_layer_comparison(results_by_layer, save_path)` -- grouped bar chart

### Step 6: src/report.py
- `generate_report(all_results)` -> markdown string
- Sections: Summary, Per-Task Results, Per-Layer Analysis, Training Dynamics, Statistical Tests, Controls

### Step 7: run.py
- Working directory guard (os.path.exists("src/benford_analysis.py"))
- Train 4 models, collect snapshots
- Run Benford analysis on all snapshots
- Generate controls (uniform, normal random)
- Save results.json and report.md
- Generate figures

### Step 8: validate.py
- Check results/results.json exists and has expected structure
- Verify all 4 models analyzed
- Verify snapshot epochs present
- Verify statistical tests computed (chi2, MAD, p-value)
- Verify figures generated
- Error accumulation pattern (collect all errors, report at end)

### Step 9: tests/
- test_data.py: shapes, value ranges, reproducibility with seed
- test_model.py: forward pass shapes, parameter counts
- test_benford.py: leading digit extraction, Benford expected values, chi2/MAD on known distributions
- test_train.py: model trains and loss decreases

### Step 10: SKILL.md
- Follow tokenizer-analysis pattern
- Steps: setup, test, run, validate, review

### Step 11: research_note/main.tex
